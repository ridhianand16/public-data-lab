# Public Data Lab

A studio for building public-facing data projects — observatories, dashboards, and explorations of open datasets.

## Structure

- **`studio/`** — how we work: philosophy, workflow, roles, naming, quality standards, and design principles.
- **`templates/`** — starting points for new projects. `observatory/` is the reference template.
- **`playbooks/`** — repeatable, end-to-end guides for common tasks.
- **`prompts/`** — reusable prompts for AI-assisted work.
- **`checklists/`** — pre-flight and review checklists.
- **`examples/`** — finished or in-progress projects worth learning from.

## Getting started

1. Read `studio/philosophy.md` and `studio/workflow.md`.
2. Copy `templates/observatory/` to start a new project.
3. Follow the relevant checklist before shipping.
