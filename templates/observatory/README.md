# Observatory Template

A starting point for a new data observatory project. Copy this folder and rename it.

## Layout

- **`docs/`** — write-ups, methodology, and source notes.
- **`decisions/`** — decision records (one file per decision).
- **`prompts/`** — project-specific AI prompts.
- **`checklists/`** — review and pre-ship checklists for this project.
- **`data/`** — raw and processed datasets.
- **`scripts/`** — data fetching, cleaning, and transformation.
- **`app/`** — the published application or site.

## First steps

1. Rename this folder to your project name.
2. Document the question and audience in `docs/`.
3. Add sources and a first decision record in `decisions/`.
